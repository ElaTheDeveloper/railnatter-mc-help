What is Towny?

Towny Basics

Towny is the plugin used on the Survival PVP Server to claim land, maintain factions and relationships.

We use the plugin Towny for optional protections for players and their stuff. How this works exactly is defined underneath here, and over at Official Towny wiki. For more information as for how to join the server or use more commands other than Towny please check out the homepage.

Nations

Nations are groups of towns with a king. The king is the mayor of the town that first created the nation, although this can be changed with the /n set king {username} command. The king, helper or assistant (you can set people as one of these as the king by typing /n rank add {username} {rank}) can add new towns to a nation, and assistants can also declare enemies and allies of the nation by typing /n enemy add {name of nation} or /n ally add {name of nation}.
